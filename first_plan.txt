📅 HAFTA 1

🗓️ Gün 1-2
📚 Temel Karın Kası Egzersizleri ve Isınma
İlk iki gün, karın kaslarını çalıştırmadan önce 5-10 dakika hafif tempolu kardiyo ile ısınma yapacağız. Ardından, temel karın kası egzersizlerine odaklanacağız. Her egzersizi 3 set 15 tekrar şeklinde yapacağız.

✅ Görevler:
• 5-10 dakika hafif kardiyo (koşu, yürüyüş)
• 3 set x 15 tekrar mekik
• 3 set x 15 tekrar ters mekik
• 3 set x 15 tekrar plank
• 3 set x 15 tekrar bisiklet mekik

🗓️ Gün 3-4
📚 Orta Şiddette Karın Kası Egzersizleri ve Esneme
Bu günlerde, zorluk seviyesini biraz artıracağız. Her egzersizi 3 set 20 tekrar şeklinde yapacağız. Egzersizlerden sonra 5-10 dakika esneme hareketleri yapacağız.

✅ Görevler:
• 5-10 dakika hafif kardiyo
• 3 set x 20 tekrar mekik (ağırlık veya direnç bandı kullanılabilir)
• 3 set x 20 tekrar yan mekik (sağ ve sol)
• 3 set x 30 saniye plank
• 3 set x 20 tekrar bacak kaldırma
• 5-10 dakika esneme

🔗 Kaynaklar:
• https://www.youtube.com/results?search_query=orta+şiddette+kar%C4%B1n+kas%C4%B1+egzersizleri
• https://www.fitnessblender.com/videos/abs-workout

🗓️ Gün 5-7
📚 İleri Seviye Karın Kası Egzersizleri ve Dinlenme
Bu günlerde, daha zorlayıcı egzersizler ekleyeceğiz. Her egzersizi 3 set 25 tekrar veya belirli bir süre boyunca yapacağız.  Yeterince dinlenmek çok önemlidir. Kaslarınızı dinlendirerek iyileşmesine izin verin.

✅ Görevler:
• 5-10 dakika hafif kardiyo
• 3 set x 25 tekrar Russian twist (ağırlık veya direnç bandı kullanılabilir)
• 3 set x 25 tekrar ayak kaldırma
• 3 set x 45 saniye plank
• 3 set x 25 tekrar  kaplumbağa mekik
• Yeterince dinlenin

🔗 Kaynaklar:
• https://www.youtube.com/results?search_query=ileri+seviye+kar%C4%B1n+kas%C4%B1+egzersizleri
• https://www.nhs.uk/live-well/exercise/strength-exercises/

📅 HAFTA 2

🗓️ Gün 8-14
📚 Hafta 1'deki egzersizleri tekrarlayın ve zorluk seviyesini artırın
İlk haftanın egzersizlerini tekrarlayın.  Tekrar sayılarını, set sayılarını veya egzersiz süresini artırarak zorluk seviyesini yükseltin.  Dinlenmeyi ihmal etmeyin.

✅ Görevler:
• Hafta 1'deki egzersizleri tekrarlayın, zorluk seviyesini artırın
• Yeterince dinlenin
• Sağlıklı beslenmeye devam edin

🔗 Kaynaklar:
